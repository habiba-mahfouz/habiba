# Habiba Mohammed - Personal Portfolio 🌸

Welcome to the repository of my personal portfolio website! This responsive, single-page application is designed to showcase my journey, technical skills, and projects as a Software Industry & Multimedia (SIM) student at Alexandria University.

## 🌟 About The Project
This portfolio features a custom **"Wildflower Meadow"** inspired design. It is built entirely from scratch to highlight my diverse skill set, serving as a centralized hub for:
- **Software Development Projects:** (Java, C++, Python)
- **Networking & Security:** (CCNA-certified configurations, Packet Tracer labs)
- **UI/UX Design:** (Wireframing, Prototyping, and UX Research)
- **Multimedia & Unity:** Audio editing and game development projects.

## 🛠️ Built With
- **HTML5 & CSS3:** Completely custom styling, CSS variables, animations, and Grid/Flexbox layouts.
- **Vanilla JavaScript:** For dynamic page transitions, navigation, and custom carousels.
- **FontAwesome:** For vector icons.
- **Google Fonts:** Utilizing 'Inter' & 'Lora' for modern typography.
